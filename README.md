📊 Bank Customer Churn Prediction

Machine Learning project to predict customer churn in the banking sector and reduce financial losses through data-driven retention strategies.

🎯 Objective

Identify customers at risk of leaving the bank and optimize retention efforts by prioritizing high-cost prediction errors (False Negatives).

🛠️ Tech Stack

Python

Pandas & NumPy

Scikit-learn

XGBoost

Matplotlib

🤖 Models Used

Logistic Regression

Decision Tree

XGBoost (Best performing model)

📈 Evaluation Metrics

Accuracy

Recall (priority metric)

Precision

F1-score

ROC-AUC

In churn prediction, missing a real churner is more expensive than a false alarm.
Therefore, Recall is prioritized.

💰 Business Perspective

Assumed costs:

False Negative = €1000

False Positive = €50

Model selection is based on financial impact, not only accuracy.

🚀 Deployment Plan

Monthly customer scoring

Risk segmentation

Targeted retention actions

🔮 Future Improvements

Deep Learning (LSTM)

Transactional data enrichment

CRM integration

📌 Conclusion

This project demonstrates how machine learning can be aligned with business strategy to reduce churn and maximize profitability.
